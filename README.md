# Memory Vault Releases

Official Windows releases for Memory Vault.

Memory Vault is a local-first desktop application for preserving, organizing, and searching family photos, documents, videos, and family-history media.

## Download

Open the **Releases** section of this repository and download the latest Windows installer.

## Privacy

Memory Vault stores your library on your own computer or external drive. Your family archive is not uploaded to this repository or to a Memory Vault cloud service.

## About this repository

This repository contains official application release files only.
